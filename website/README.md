# Get Started in 5 Minutes

1. Make sure all the dependencies for the website are installed:

```sh
# Install dependencies
$ npm install
```
2. Run your dev server:

```sh
# Start the site
$ npm start
```

## Directory Structure

Your project file structure should look something like this

```
tartiflette.io/
  docs/
    api/
      directive.md
      engine.md
      ..
    tutorial/
      add-subscription.md
      ...
    ...
  website/
    core/
    node_modules/
    pages/
    static/
      css/
      img/
    package.json
    sidebar.json
    siteConfig.js
```
